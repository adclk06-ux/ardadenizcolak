# Arda Deniz Çolak

Digital platform for Arda Deniz Çolak.

Built with:
- Astro
- TypeScript
- pnpm

Documentation lives in `/docs`.

## Environment Variables

| Variable | Description |
|---|---|
| `PUBLIC_CONTACT_FORM_ACCESS_KEY` | Web3Forms or Formspree access key for the contact form. If unset, the form renders but submissions will fail. |

Example `.env`:
```
PUBLIC_CONTACT_FORM_ACCESS_KEY=your_web3forms_key_here
```
